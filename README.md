# Gerenciador de Veículos

Um aplicativo mobile desenvolvido em React Native para o gerenciamento de veículos, com dados sincronizados e persistidos na nuvem através do Firebase.

## Sobre o Projeto

O objetivo deste aplicativo é facilitar o controle de veículos. Após criar uma conta e fazer login, o usuário tem acesso a um painel onde pode administrar seu perfil e realizar o gerenciamento completo (CRUD) de seus veículos com uma interface simples e direta.

## Funcionalidades

- **Autenticação:** Sistema de login e cadastro de novos usuários.
- **Perfil do Usuário:** Área dedicada para visualização e gerenciamento das informações da conta.
- **Gestão de Veículos (CRUD):**
  - **Criar:** Cadastro de novos veículos no sistema.
  - **Consultar:** Listagem de todos os veículos cadastrados.
  - **Atualizar:** Edição de informações dos veículos já existentes.
  - **Excluir:** Remoção de veículos do banco de dados.
- **Sincronização em Nuvem:** Dados salvos e atualizados remotamente utilizando o Firebase Firestore.

## Tecnologias Utilizadas

- [React Native](https://reactnative.dev/): Framework base para o desenvolvimento.
- [Expo](https://expo.dev/): Plataforma para facilitar a construção e testes do app.
- [Firebase Authentication](https://firebase.google.com/docs/auth): Gerenciamento de login e cadastro.
- [Firebase Firestore](https://firebase.google.com/docs/firestore): Banco de dados NoSQL remoto.
