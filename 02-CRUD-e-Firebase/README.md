# CRUD e Firebase
Nesta etapa foi realizada a integração do aplicativo com o Firebase, utilizando o Firebase Authentication para autenticação de usuários e o Cloud Firestore para armazenamento remoto dos dados.

CRUD completo de veículos:

- Create: cadastro de veículos contendo marca, modelo, ano, placa e número do chassi.
- Read: listagem dos veículos cadastrados.
- Update: edição das informações dos veículos.
- Delete: exclusão de veículos cadastrados.
- Persistência remota dos dados utilizando Cloud Firestore.
- Navegação entre as telas utilizando React Navigation.
